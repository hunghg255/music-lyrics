## Install

```bash
npm i @hunghg255/music-lyrics@latest
```

With `yarn`

```bash
yarn add @hunghg255/music-lyrics@latest
```

## Basic use

```js
import { SearchLyrics } from '@hunghg255/music-lyrics';

// Node
// const { SearchLyrics } = require('@hunghg255/music-lyrics');
```
